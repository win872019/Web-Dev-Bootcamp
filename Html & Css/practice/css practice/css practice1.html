向日葵

向日葵（common sunflower，學名：Helianthus annuus）是菊科向日葵屬一年生闊葉草本（forb）。 別名太陽花、朝陽花、日頭花[3]。因花序隨太陽轉動（太陽花）而得名，花語為太陽。


形態

一年生草本 。莖直立，圓形多稜角，質硬白色粗硬毛。 廣卵形的葉片通常互生，先端銳突或漸尖，有基出3脈，邊緣具粗鋸齒，兩面粗糙，被毛，有長柄。頭狀花序，直徑9.5～30公分，單生於莖頂或枝端。 總苞片多層，葉質，覆瓦狀排列，被長硬毛，花期為7到8月，花序邊緣生中性的黃色舌狀花，不結實。 花序中部為兩性管狀花，棕色或紫色，能結實。矩卵形下位瘦果，果皮木質化，灰色或黑色，稱葵花籽 。



大小

最常見的向日葵高度為2.5～3.5公尺。根據1567年的科學文獻報告，歷來被認為單頭向日葵植物是種植在帕多瓦，植株高度最高可達12公尺。 同樣的種子，種植在其他時間和地點（如馬德里），植株高度可達近8公尺。更近的紀錄（約二十年前）則有8公尺以上的植株，在荷蘭和加拿大安大略省。 向日葵高約3公尺（9.8英尺），最高可達9.17公尺（30.1英尺）[4]。


習性

向日葵會隨著太陽的移動轉動花盤。


象徵

向日葵是烏克蘭的國花。在2022年俄國入侵烏克蘭期間，社交媒體上廣為流傳的一個影片中，一名不知名的烏克蘭婦女叫侵烏俄軍士兵「拿上這些向日葵種子，把它們裝在口袋裡。 這樣向日葵就會在你們死掉的地方生長」[5]。向日葵從此在烏克蘭和全世界成為了抵抗侵略以及團結和希望的象徵[6]。
